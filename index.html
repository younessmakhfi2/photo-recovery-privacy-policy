1. Introduction

Qsoft LLC built the File Recovery & Photo Recovery app as a free tool to help users recover deleted photos and files. This app is provided at no cost and intended for use as-is.

2. Information Collection and Use

We do not collect, store, or share any personal or user data. All file recovery operations occur locally on your device. No data is transmitted to any server.

3. Permissions Requested

To function correctly, the app requires the following permissions:

  • READ/WRITE_EXTERNAL_STORAGE: Access deleted files to restore them.
  • MANAGE_EXTERNAL_STORAGE: Full access to directories (for Android 11+).
  • INTERNET / ACCESS_NETWORK_STATE / ACCESS_WIFI_STATE: Needed for ads and connectivity checks.
  • AD_ID: Used by AdMob for personalized advertising.

4. Purpose of Data

The above permissions are solely used to scan and recover files. No user data is collected, stored, or shared.

5. User Rights

Since we don’t store or collect user data, there’s nothing to delete or manage. All processes are local.
